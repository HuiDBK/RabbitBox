# RabbitBox
中秋特辑-🐰兔子吃着月饼🥮上月球🌕


## 项目结构
```bash
├── res # 游戏素材
│   ├── img
│   │   ├── bg_blue.png
│   │   ├── bg_girl.jpeg
│   │   ├── bg_嫦娥_gray.png
│   └── music
│       ├── 卡农.mp3
│       └── 龙珠.mp3
└── src
    ├── game_main.py # 游戏入口
    ├── game_map.py # 游戏地图
    └── game_setting.py # 游戏配置
├── LICENSE
├── README.md
├── main.py # 主入口
├── requirements.txt
```

## 依赖安装
```python
pip install -r requirements.txt
```
